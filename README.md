# scanner

Scanner is script  with cpp applicable to scan network ranges like advanced ip scanner and ip angry 
charactrized by the speed in scan.

### compile

```
make 

```
### run

Usage: scanner <speed> [1][2][3]

```
scanner 2
  
```
